## Sass 
* Syntactically Awesome Style Sheets
* CSS 전처리기 
* 복잡한 작업을 쉽게 할 수 있도록 도와주고, 스타일 코드의 재활용성과 코드의 가독성을 높여 유지보수를 더욱 쉽게 해준다 
* 가능한 확장자 
    * Sass
    ```css
    $font-stack: Helvetica, snas-serif
    $primary-color: #333

    body 
        font: 100% $font-stack;

    ```
    * Scss
    ```css
    $font-stack: Helvetica, snas-serif;
    $primary-color: #333;

    body {
        font: 100% $font-stack;
    }
    ```
